# artale-mob-n-shot-calculator

A simple calculator for MapleStory Artale, simulating skill damage distribution and calculating how many hits to kill a mob with certain skill combos.

## Features

- Job and weapon type selection
- Stat input for STR, DEX, INT, LUK
- Attack range input with WATK reverse calculation
- Buff support: Takoyaki, Snowflake Crystal, custom WATK buffs
- Monster selection with HP and defense info
- Skill selection with per-level damage data
- Venom support with configurable level
- Monte Carlo damage simulation (100,000 iterations)
- Kill count distribution histogram
- Detailed per-hit damage breakdown
- Responsive 2-column layout (1-column on mobile)
