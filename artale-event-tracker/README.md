# artale-event-tracker

MapleStory Artale TW server event tracker for daily/weekly/one-time events and shop exchange planning.

## Features

- Multiple event support with tab switching
- Daily, weekly, and one-time task tracking
- Auto-reset timers for daily and weekly tasks
- Task completion progress display per section
- Shop exchange calculator with item quantities
- Collapsible sections for completed task groups
- Past record editing for missed days
- Event history log with daily earn/spend breakdown
