# artale-exp-time-calculator

A simple web-based calculator for MapleStory Artale players to estimate the time needed to reach their target level based on current EXP efficiency.

## Features

- Simple & Advanced modes for calculation
- EXP calculation from current level to target level
- Time estimation based on EXP efficiency (per 10 minutes)
- Dual unit input: units of 10,000 or regular integers
- Current level progress percentage display
- Regular buffs: Holy Symbol (10%/15%/20%) and Showdown (+10%)
- Event support: EXP coupons (2x/3x), weather events (2x/3x), custom events
- Event time breakdown: phase-by-phase duration details
- Grinding schedule: daily time mode and target days mode
- Level history tracking with visual line chart
- Smart daily aggregation for long-term history
- Export/Import records as text files
