# Artale Tool Sets

A collection of web tools for MapleStory Artale players, hosted as a single GitHub Pages site.

## About

MapleStory Artale is an MMORPG game. This site provides a unified collection of calculators and trackers to help players plan their gameplay more efficiently.

## Tools

- EXP Time Efficiency Calculator - estimate grinding time to reach a target level, with support for buffs, events, schedules, and level history tracking
- Event Tracker - track daily/weekly/one-time event task progress and shop exchange planning for TW server events
- Mob N-Shot Calculator - simulate skill damage distribution and calculate how many hits to kill a mob with certain skill combos

### Data Persistence

All tools automatically save your inputs to browser localStorage. No personal data is sent over the internet.

### Themes

- Dark Theme (Default)
- Light Theme

Toggle between themes using the button in the top-right corner. Theme preference syncs across all tools.

### Browser Compatibility

Works on all modern browsers that support:

- ES6 JavaScript
- CSS Custom Properties
- LocalStorage API
- HTML5 Form Validation
